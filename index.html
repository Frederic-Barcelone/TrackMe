TrackMe by wireman

Real-time GPS tracking system, with social features on top of it.

Has a web-version and a mobile app. For now, the mobile app will support only Android-based platforms.

Install

To install this project, you should have Rails 2.3.5 installed and a supported by Rails database, e.g. PostgreSQL or SQLite.
